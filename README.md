# User-Adder-List
A complete Spring Boot MVC user registration project featuring controller-layer architecture, Thymeleaf templates, and H2 database integration. Includes CRUD operations and basic form validation.
After you download whole files open and run src/main/java/com/hidircan/spring/Application.java and then you need to use a web browser which yoou want and you need to write localhost:7001/users and then you can create lists.
Theese lists will be stored in datafolder which is will created after your run.
